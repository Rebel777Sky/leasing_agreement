**Внешняя обработка «Добавить запись из внешнего источника»**

За определенный период позволяет на стороне **1С: УНФ**, получить из **1С: ERP** договора контрагентов.

Связь между решениями ИБ через механизм «COMConnector», объект для версии 8.3 используем объект «V83.COMConnector» соответственно.
В своем проекте для установки связи использую метод **БСП** «УстановитьВнешнееСоединениеСБазой()»,  расположен в «ОбщийМодуль.ОбщегоНазначенияСлужебныйКлиентСервер».
