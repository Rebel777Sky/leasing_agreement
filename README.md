# Внешняя обработка «Добавить запись из внешнего источника»

## Требование

*Используйте платформу 1с не ниже версии 8.3.27.1688*

БСП "Стандартные подсистемы" версия не ниже 3.1.10.309, посмотрите в регистре сведений "Версии подсистем".

## Описание
За определенный период позволяет на стороне **1С: УНФ**, получить из **1С: ERP** договора контрагентов.

Связь между решениями ИБ через механизм «COMConnector», объект для версии 8.3 используем объект «V83.COMConnector» соответственно.

В своем проекте для установки связи использую **БСП** метод «УстановитьВнешнееСоединениеСБазой()»,  расположен в «ОбщийМодуль.ОбщегоНазначенияСлужебныйКлиентСервер».
